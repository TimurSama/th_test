# Запуск бота на локальном ПК для демонстрации

## ✅ Да, можно использовать свой ПК!

Партнеры смогут открыть бота в Telegram и использовать его, пока ваш ПК включен и подключен к интернету.

## Быстрый запуск

### 1. Убедитесь, что Python установлен

```bash
python --version
```

Должно быть Python 3.11 или выше.

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Создайте .env файл

Создайте файл `.env` в корне проекта со следующим содержимым:

```
TELEGRAM_BOT_TOKEN=8591869020:AAFnlsirUwd3TKMibAWDE209OBwVU40ZEjo
DATABASE_PATH=./data/tokenhunter.db
EXCHANGES=binance,okx,bybit,gateio
```

Или используйте скрипт:
```bash
python scripts/setup_env.py
```
(Введите токен когда попросит)

### 4. Запустите бота

```bash
python main.py
```

Бот запустится и будет работать пока вы не остановите его (Ctrl+C).

## Для демонстрации партнерам

1. **Запустите бота на своем ПК** (команда выше)
2. **Найдите бота в Telegram** по username (если задан) или по токену
3. **Партнеры могут:**
   - Открыть бота в своем Telegram
   - Отправить `/start`
   - Использовать все функции бота
   - Видеть данные в реальном времени

## Важно для демонстрации

### ✅ Что работает:
- Бот отвечает на команды
- Все функции доступны
- Данные обновляются (если GitHub Actions работает)
- Партнеры видят то же самое, что и вы

### ⚠️ Ограничения:
- Бот работает только пока ваш ПК включен
- Если ПК заснет - бот перестанет отвечать
- Нужен стабильный интернет

## Как сделать демонстрацию стабильной

### Вариант 1: Предотвратить сон ПК

**Windows:**
- Настройки → Система → Питание
- Установите "Никогда" для сна при питании от сети

**Mac:**
- Системные настройки → Энергосбережение
- Отключите сон

### Вариант 2: Использовать screen/tmux (если Linux/Mac)

```bash
screen -S bot
python main.py
# Нажмите Ctrl+A затем D для отсоединения
```

### Вариант 3: Запустить как службу Windows

Можно создать задачу в Планировщике задач Windows для автозапуска.

## Проверка работы

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Бот должен ответить приветственным сообщением

Если бот не отвечает:
- Проверьте, что `python main.py` запущен
- Проверьте логи в консоли на ошибки
- Убедитесь, что `.env` файл существует и содержит токен

## Логи

В консоли вы увидите:
```
INFO - Initializing database...
INFO - Database initialized
INFO - Starting TokenHunter bot...
INFO - Bot starting...
```

После этого бот готов принимать команды!

## Остановка бота

Нажмите `Ctrl+C` в консоли где запущен бот.
